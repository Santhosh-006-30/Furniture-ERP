# Feature Matrix
**Shiv Furniture Works Mini ERP — Operations & Observability Matrix**

| Module | Features Implemented | Access Roles |
| :--- | :--- | :--- |
| **Customer Portal** | Register, Login, Catalog, Search/Filters, Compare, Wishlist, Cart, Checkout, Invoices, Return request. | Customer |
| **Sales Management** | Draft Sales Orders, Customer Invoicing, Payment Verification, Order Dispatch, Delivery Tracking. | Admin, Owner, Sales |
| **Manufacturing** | Bill of Materials (BOM), Work Center Capacities, Manufacturing Orders (MO), Work Orders scheduling. | Admin, Owner, Manufacturing |
| **Procurement** | Make-to-Stock (MTS) & Make-to-Order (MTO), Vendor Ratings, Low-stock PO draft generation. | Admin, Owner, Purchase |
| **System Admin** | OS memory footprint, DB size, DB latency checks, Cron background jobs monitoring. | Admin, Owner |
| **Security & Audits** | Rate limiting, CSP, HSTS, XSS sanitization, JSON diff Audit Log explorer. | Admin, Owner |
| **Notifications** | In-app notification center categorizing alerts by priority (Low, Medium, High). | All Roles |
